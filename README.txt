CSE430 Project 3 Part 1
by Daniel Caruso II, Josh Monson, and Adam Mastov
Description: Program walks through the page table
of a process given its PID and virtual address as paramaters.
If the process is in physical memory, prints the
physical address of the process memory, if the page
of the process is in swap, prints the swap identifier,
or, if the memory is unavailable, prints that the memory
is unavailable.
